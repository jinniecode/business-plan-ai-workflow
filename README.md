# Business Plan AI Workflow

## 목적

PRD, 기능정의서, 정책제안서, 메뉴설계서를 바탕으로  
AI를 활용해 사업기획서 작성 워크플로우를 구성한다.

## 사용 AI

- Codex: 문서 통합, 워크플로우 자동화, 중간 산출물 생성
- Gemini: 시장, 고객, 경쟁 근거 보강
- ChatGPT: 사업기획 논리 정리
- Claude: 심사자 관점 검토
- Genspark: 최종 발표자료 제작

## 최종 목표

1시간 이내 스터디에서 기존 기획 산출물을  
사업기획서 발표자료로 전환하는 AI-native 워크플로우를 실습한다.
